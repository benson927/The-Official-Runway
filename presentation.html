什麼是
AGENT SKILLS?

在 Google Antigravity 框架下，Agent Skill 是 AI 代理人的「專家擴充知識庫」。AI 透過讀取專案中定義的 Skill 設定與 Markdown 規則，能瞬間獲取特定領域的最新最佳實踐、效能指引與常見避坑指南。

03 /

技術痛點與
GSAP SKILLS

應用場景：常規開發中，AI 代理人預設容易寫出舊版 GSAP 語法，或是忘記在 React 元件卸載時調用 gsap.context() / 進行 Cleanup。這會導致記憶體洩漏與重複渲染引起的「Layout Thrashing」（佈局抖動），使時尚網格動畫嚴重卡頓。

解決方案：我們在專案根目錄注入了 .gsap-rules.md 與 GSAP React 專家規則，強迫 AI 嚴格使用 useGSAP() hook 進行動畫註冊，達成「官方級」最優渲染效能。

[ X ] 傳統 AI 容易犯的錯誤 
// 缺點：
// 1. 忘記在 React 卸載時做 cleanup
// 2. 舊版 GSAP 寫法，可能重複觸發動畫
useEffect(() => {
  gsap.to(".look-card", {
    opacity: 1,
    y: 0,
    stagger: 0.1,
    duration: 1
  });
}, []); // ❌ 重複渲染時會導致動畫堆疊與記憶體洩漏
嚴重度：高 (導致時尚網格掉影格、發熱)
[ ✓ ] GSAP-React Skill 規範 
// 優點：
// 1. 使用官方 @gsap/react 專用 Hook
// 2. 自動在組件 unmount 時清理
useGSAP(() => {
  gsap.to(".look-card", {
    opacity: 1,
    y: 0,
    stagger: {
      each: 0.1,
      ease: "power2.inOut"
    },
    duration: 1,
    clearProps: "all" // 確保動畫結束後清除行內樣式
  });
}, { scope: containerRef }); // ✓ 作用域限定，自動 Cleanup
結果：60FPS 流暢渲染，完全零記憶體殘留

機器學習核心
THE ORACLE

The Oracle Vision API 是系統內嵌的機器學習引擎。當後端爬蟲取得 Vogue Runway 時裝影像後，多模態 ML 模型會對照片進行即時的「時裝語意解析」，並自動生成包含材質、剪裁、色彩與年份的結構化標籤。

這徹底廢除了繁瑣的資料夾管理，直接依賴多維度標籤在 SQLite 與 Supabase 中進行秒級資料檢索。

ML 視覺呈現：
暗房顯影

The Darkroom Reveal：為了展現時裝的歷史感與光學物理質地，我們摒棄了生硬的網頁淡入，而在 GSAP 專家 Skill 的引導下實作了「光學顯影」動畫。

影像會模擬底片在化學藥水中的顯影過程：從高度光學模糊 (Blur 30px)、低對比與高曝光，經過骨牌式序列（Stagger），在 0.8 秒內自動「光學對焦」至極度銳利的原始影像，流暢度高達 60fps。
